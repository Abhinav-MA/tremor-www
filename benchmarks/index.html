Tremor Benchmarks

Here you can track "live" benchmarks of the tremor runtime for every PR that is merged into the main branch. The benchmarks are run on the same machine to allow them to be compared easiley.
This allows comparing and tracking the performance progression of tremor over time. Clicking on a node will show the related commit hash.
